    Check yo self before you wreck yourself
      2014 Fall Fuse "Heath-care at Home" Hackathon • Dublin, Ohio
      • "Check yo self before you wreck yourself" was a phone app that we developed at the 2014 Fall Fuse "Heath-care at Home" Hackathon. This was basically an aggravate, that would get information from three sources. WebMD, Mayo Clinic, and Wikipedia, and would display the symptoms and treatments for the ailment the user entered.
      • Github repository

    Event Findr
      2016 PennAppsXIII • Philadelphia, Pennsylvania
      • "Event Findr" was a website that attempted to compile a list of college events, that are open to the public, into a single centralized place that the public could easily access. To achieve this we pulled the information from the colleges RSS feeds and extracted the information contained in the XML files. We then made the information searchable by category, city, and date.
      • Github repository

    Walmart trips
      MHacks 6 • Ann Arbor, Michigan
      • Some experimentation with the Walmart API that I did during MHacks 6. The idea was to show the user the furthest Walmart store away from them, and then use the Expedia API to book a flight to it. While I was not able to complete this project due to technical difficulties, this project advanced my knowledge of both how APIs work as well as javascript.
